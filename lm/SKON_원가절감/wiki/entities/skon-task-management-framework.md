# SKON 과제 관리 체계

SKON의 원가절감 활동을 체계적으로 관리하기 위한 분류 체계입니다. 이 체계는 과제의 성격, 중요도, 난이도, 진행 상태 등을 다차원적으로 분석할 수 있도록 설계되었습니다.

## 과제유형 (Task Type)

원가절감 활동의 성격을 구분하는 기준입니다. 총 10개 유형으로 분류됩니다.

| 과제유형 | 설명 |
|---------|------|
| 구매원가절감 | 소재, 부품 등의 구매 단가 인하 활동 |
| 설비효율화 | 설비 가동률 개선, 유지보수 최적화 |
| 간접비절감 | 간접재 구매 통합, 관리비 절감 |
| 물류비절감 | 운송 경로 최적화, 포장 경량화 |
| 에너지절감 | 에너지 사용 효율화, 재생에너지 전환 |
| 재고최적화 | 재고 수준 적정화, 보관비 절감 |
| 공정개선 | 생산 공정 자동화, 효율화 |
| 품질비용절감 | 불량률 저감, 클레임 비용 절감 |
| 외주비절감 | 외주업체 정예화, 내재화 |
| 인건비효율화 | 생산 자동화, 인력 효율화 |

## 카테고리 (Category)

절감되는 비용의 종류를 나타냅니다. 총 10개 카테고리로 분류됩니다.

| 카테고리 | 설명 |
|---------|------|
| 소재비 | 원자재, 주요 소재 비용 |
| 물류비 | 운송, 보관, 포장 관련 비용 |
| 부품비 | 구매 부품, 공용 부품 비용 |
| 품질비 | 불량, 클레임, 검사 관련 비용 |
| 관리비 | 간접재, 운영비, 관리비 |
| 인건비 | 직접 인건비, 생산 인력 비용 |
| 외주비 | 외주 가공, 외주 업체 비용 |
| 가공비 | 내부 가공, 설비 가공 비용 |
| 유틸리티 | 에너지, 전력, 수도 등 |
| 설비감가 | 설비 감가상각비, 유지보수비 |

## 과제상태 (Task Status)

과제의 진행 단계를 나타냅니다.

| 상태 | 설명 |
|------|------|
| 계획 | 과제가 계획 단계에 있음 |
| 검토중 | 과제의 타당성 및 효과를 검토 중 |
| 진행중 | 과제가 실행 중 |
| 보류 | 과제가 일시 중단됨 |
| 완료 | 과제가 완료됨 |

## 우선순위 (Priority)

과제의 중요도를 나타냅니다.

| 우선순위 | 설명 |
|---------|------|
| 상 | 전략적 중요도가 높고, 즉시 실행 필요 |
| 중 | 중요하지만 일정 조정 가능 |
| 하 | 점진적 개선 과제 |

## 난이도 (Difficulty)

과제 수행의 어려움 정도를 나타냅니다.

| 난이도 | 설명 |
|-------|------|
| 고 | 기술적, 조직적 장벽이 높음 |
| 중 | 일정 수준의 노력과 자원 필요 |
| 저 | 비교적 쉽게 실행 가능 |

## 절감방식 (Reduction Method)

구체적인 원가절감 방법을 나타냅니다.

| 절감방식 | 설명 |
|---------|------|
| 통합발주 | 구매 물량 통합을 통한 단가 인하 |
| 단가 재협상 | 기존 공급사와 단가 재협상 |
| 공용화 설계 | 부품, 소재의 공용화 |
| 공정 자동화 | 생산 공정 자동화 |
| 벤더 통합 | 공급사 통합 및 정예화 |
| 대체소재 적용 | 저가 대체 소재 적용 |
| 패키지 경량화 | 포장재 경량화 |
| 경로 최적화 | 물류 경로 재설계 |
| 설비 PM 주기 조정 | 설비 예방 정비 주기 최적화 |
| 에너지 모니터링 | 에너지 사용량 모니터링 및 절감 |
| 수요예측 정밀화 | 수요 예측 정확도 개선 |
| 재고감축 | 재고 수준 최적화 |
| 리사이클 적용 | 재활용 소재 적용 |
| 내재화 | 외주 가공의 내부화 |
