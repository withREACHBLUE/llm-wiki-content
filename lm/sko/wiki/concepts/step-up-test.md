# Step-up Test

## 정의
표준 속도로 회귀한 후 5% 단위로 단계적으로 속도를 증가시키며 데이터 로그를 분석하는 체계적인 테스트 방법.

## BlueOval SK 사례
- 코팅 속도 110% 상향 시도 실패 후, 표준 속도 회귀 후 Step-up Test를 통해 최적 속도 도출 결정.
- 데이터 기반 의사 결정의 구체적 사례.
