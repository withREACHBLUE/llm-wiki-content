# DT (Digital Transformation) 연계

## 정의

디지털 기술을 활용하여 업무 프로세스를 혁신하고 데이터 기반 의사 결정을 지원하는 것.

## 적용 사례

- **Tableau 대시보드:** 실시간 원가 분석 대시보드 고도화
- **Daily Cost Report 자동화:** 각 공장별/라인별 변동비를 매일 아침 경영진에게 자동 리포팅
- **이메일 알림 시스템:** 자동화된 알림 기반의 보고 체계 구축

## 효과

원가 관리의 속도와 정확성 향상, 신속한 의사 결정 지원.
