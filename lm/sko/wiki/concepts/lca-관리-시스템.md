# 전사 통합 LCA 관리 시스템

## 개요
전과정 평가(Life Cycle Assessment) 방법론을 기반으로 배터리 전 생애주기의 탄소 발자국을 통합 관리하는 시스템. EU 배터리법 대응을 위해 신규 구축이 결정된 프로젝트.

## 주요 기능
- 원재료 채굴부터 폐기·재활용까지 전 단계 탄소 배출량 측정 및 관리
- DPP 데이터 연동을 위한 표준화된 데이터 인터페이스 제공
- 협력사 데이터 수집 및 검증 자동화
- 규제 보고서 자동 생성

## 프로젝트 현황
- 구축 프로젝트 착수 결정 완료
- 데이터플랫폼팀이 기술 리드
- DPP 프로토타입과의 데이터 연동 방안 설계 중
